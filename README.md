# Arita

Project with custom animated slider used GreenSock Animation Platform

## Description

That project developed in order to studying GSAP and practice with browser events.

Main feature:
1. Custom slider with GSAP timeline animation;
2. Custom cursor;

## Stack

* HTML;
* CSS;  
* JavaScript;
* GSAP;
